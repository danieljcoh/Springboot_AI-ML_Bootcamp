# How to Run:
You need to add these instructions to your repository README.md to get full points.

## Installation & Setup
- Prerequisites: Ensure pokemon.csv and pokemon_battle_model.pkl are in the folder.

## Install requirements:
- `pip install -r requirements.txt`

## Run the Server
- `uvicorn main:app --reload`
- View UI: Open your browser to http://127.0.0.1:8000/docs.
- Run Demo: Open demo_api.ipynb and run the cells.
